# 0ne Skills — Claude Code Plugin Marketplace

Premium AI skills for Claude Code by [0neOS](https://0neos.com).

## Install

Add this marketplace to Claude Code:

```
/plugins marketplace add https://github.com/becomingjimmy/0ne-plugins
```

Then browse available skills in the Plugins > Discover tab.

## Activation

All skills require a 0ne subscription. After installing a plugin, the first time you use it you'll be guided through activation:

1. Sign up at [0neos.com](https://0neos.com)
2. Install 0ne (includes your API token)
3. Activate individual skills via the CLI
